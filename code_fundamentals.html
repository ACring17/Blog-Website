APIs

While watching Youtbe I came across a great video from FreeCodeCamp that teaches beginners about application programming interfaces (API). As the video explains, APIs are the functions of programs that the user learns to interact with. APIs are used to make things easier and take the away the complication of what's happening behind the scenes. Like a radio, the user knows how to work the power button, volume and station controls, but has no idea how the inner workings of the radio does that. APIs work the same but for making code.

People have developed shortcuts to make functioons easier to add to websites and allow developers to get creative with already existing code. The video was very informative and has shown me a world of possibilities to add to my coding skills. After watching this video I want to incorporate a project that uses some-sort of web API and create something fun. I am thinking of using Spotify's API because I am a big fan of the platform. Now I just need to think of a clever idea to incorporate it with. I listed the link to the freecodecamp video for reference.

Build to learn

Mar. 29, 2021

I have spent the last six months learning HTML, CSS, and JavaScript and once of the best ways for me to absorb and understand how these languages work is to practice using the lessons that I have learned. I have found that tinkering with the newly learn code really helps me understand how it works and the limitations to the abilities of the language. If I do not put in the physical practice of using the new code I find that I do not retain the lessons as welll as compared to the lessons I practice and implement into my projects.

Like any skill learning to code requires practice and repetition to create the neural connections in the brain. The more routine it becomes the easier it will become to master. I am really focusing on coding everyday so that I get a little bit better and can understand how this new complex world of software works. My next goal is to learn and understand React.JS. I believe this will help further my understanding of JavaScript and make one step closer to employable in front-end development.

Coding Fundamentals: Consistency is Key

Feb. 17, 2021

With coding fundamentals I will be talking about key lessons learned while in my coding journey. Ideas and topics that will be helpful to revisit and teach other learn lessons that are important to me. The first one will be about the lesson of perseverance and consistency. These lessons and skills are not only important for coding but in life in general.

I have found that coding is like fitness, in that to become great and be the envy of everyone in the field is a long journey of working hard and being consistent. To see results with weight training one must be consistent with lifting heavy weights on a regular baisis. To be able to run a marathon one must have regular running routine. Similarly, I have learned that to learn code one must find a routine and practice regularly. I have found that when I am consistent with my coding practice the skills seem to stick in my head and become natural to me. I create a new route for my brin to function.

When I take a break or lose motivation from coding, I find it to be hard to get back into the rythm of learning and practicing. Through just getting a little bit done for coding I can use the 1% of getting better and let it grow over time. That 1% will continue to grow each day and compound into mastery of the language. The goal should not be to rush through the lessons and learn the concepts fast, but to truly learn the lesson and take my time learning the concepts. When I feel lack of motivation, I need to remember that action will help show me success, which will in turn help me create motivation and momentum in learning. Even if you feel dumb, keep learning and practicing. Eventually it will amount into mastery!
