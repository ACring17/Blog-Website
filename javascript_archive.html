Learning JavaScript

Jan. 22, 2021

JavaScript has proven to be the hardest of the three languages of front end development. I gained a lot of confidence learning the mark-up languages of HTML and CSS where it is pretty straight forward witht the commands and inputs. While JavaScript seems to be more computational, it seems my hardest lesson is learning and understanding the best way to utilize JavaScript. I need to be better about my practice time with JavaScript and make it more of a daily habit till i fully grasp it. It can be a bit overwhelming at times and makes me feel fatigue. It is for sure the first language that makes me feel the imposter syndrome that comes with programming.

Feeling extra motivated to learn JavaScript this year and start my career in coding. I feel like I am so close to getting over the learning curve of JavaScript. With continued practice I know I will one day eventually master the front end languages. Once I master JavaScript I know it will open the door to many programming and tech opportunities. I like the challenge and cannot wait to taste the sweet flavor of success.
