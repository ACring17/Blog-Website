User Interaction: Alert, Prompt, & Confirm

Mar. 21, 2021

The browser gives us a few functions to interact with our users. Alert will display a message on the screen. The user must acknowledge the window before moving onto the rest of the page by pressing ok. Next is the Prompt function, which takes two arguments and can ask a user a question and allows the user to input an answer. And lastly we have the Confirm function which will give the user a question and two options: Ok or cancel. This will tanslate into True if OK is selected and False if canceled is selected.

All three functions are handy to have for big website design and can be used as tools to help test or debug a site. They are fairly disruptive on a web page and should be lmited in their use. But for instances of purchases and big desions for the user these options can be usefulto help guide the user. I am looking forward to learning more useful functions to help me master JavaScript.
